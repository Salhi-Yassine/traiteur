# Symfony Entity Anatomy

Canonical PHP attribute ordering and conventions for Farah.ma entities. Derived from `api/src/Entity/VendorProfile.php`.

---

## Class-level attribute order

Always in this sequence:

```php
#[ApiResource(
    operations: [
        new GetCollection(),
        new Get(),
        new Post(security: "is_granted('entity:create')"),
        new Patch(security: "is_granted('entity:edit', object)"),
        new Delete(security: "is_granted('ROLE_ADMIN') or is_granted('entity:edit', object)"),
    ],
    normalizationContext: ['groups' => ['entity:read']],
    denormalizationContext: ['groups' => ['entity:write']],
    order: ['createdAt' => 'DESC'],  // only if ordering matters
)]
#[ApiFilter(SearchFilter::class, properties: ['field' => 'exact', 'relation.slug' => 'exact'])]
#[ORM\Entity(repositoryClass: MyEntityRepository::class)]
#[UniqueEntity(fields: ['slug'], message: 'This slug is already taken')]  // only if applicable
```

---

## Property order inside the class

1. **Class constants** (enum-like values)
2. **id** — generated, read-only
3. **slug** — auto-generated by Gedmo, read-only in practice
4. **Translatable string fields** — businessName, tagline, description (with `#[Gedmo\Translatable]`)
5. **Regular scalar fields** — other strings, integers, booleans, JSON arrays
6. **ManyToOne relations** — before OneToMany
7. **ManyToMany relations**
8. **OneToOne relations**
9. **OneToMany collections** — last among relations
10. **Computed / system fields** — averageRating, reviewCount, isVerified, isFeatured
11. **Timestamps** — createdAt, updatedAt (always last)

## Standard property templates

```php
// Id — always exactly this, never add write group
#[ORM\Id]
#[ORM\GeneratedValue]
#[ORM\Column]
#[Groups(['entity:read'])]
private ?int $id = null;

// Slug — auto-generated, expose in read only
#[ORM\Column(length: 255, unique: true)]
#[Gedmo\Slug(fields: ['name'])]
#[Groups(['entity:read'])]
private ?string $slug = null;

// Translatable string — fr stored in column, ar/ary/en in ext_translations
#[ORM\Column(length: 255)]
#[Assert\NotBlank]
#[Gedmo\Translatable]
#[Groups(['entity:read', 'entity:write'])]
private string $name = '';

// Nullable translatable string
#[ORM\Column(length: 255, nullable: true)]
#[Gedmo\Translatable]
#[Groups(['entity:read', 'entity:write'])]
private ?string $tagline = null;

// Gedmo locale hint — NOT a database column
#[Gedmo\Locale]
private ?string $locale = null;

// ManyToOne relation
#[ORM\ManyToOne(targetEntity: Category::class, inversedBy: 'myEntities')]
#[ORM\JoinColumn(nullable: true)]
#[Groups(['entity:read', 'entity:write'])]
private ?Category $category = null;

// Computed/system field — read-only
#[ORM\Column(type: 'decimal', precision: 3, scale: 2, nullable: true)]
#[Groups(['entity:read'])]
private ?string $averageRating = null;

// Timestamps — always last
#[ORM\Column(type: 'datetime_immutable')]
#[Gedmo\Timestampable(on: 'create')]
#[Groups(['entity:read'])]
private \DateTimeImmutable $createdAt;

#[ORM\Column(type: 'datetime_immutable', nullable: true)]
#[Gedmo\Timestampable(on: 'update')]
#[Groups(['entity:read'])]
private ?\DateTimeImmutable $updatedAt = null;
```

## Enum-like constants pattern

Derived from `VendorProfile`:

```php
class MyEntity
{
    public const STATUS_PENDING  = 'pending';
    public const STATUS_APPROVED = 'approved';
    public const STATUS_REJECTED = 'rejected';

    #[ORM\Column(length: 20)]
    #[Assert\Choice(choices: [self::STATUS_PENDING, self::STATUS_APPROVED, self::STATUS_REJECTED])]
    #[Groups(['entity:read'])]
    private string $status = self::STATUS_PENDING;
```

## Serialization group rules

| Field type | Groups |
|---|---|
| Auto-generated id | `['entity:read']` only — **never** write |
| Auto-generated slug | `['entity:read']` only — **never** write |
| User-editable field | `['entity:read', 'entity:write']` |
| Computed/system field (rating, count, verified) | `['entity:read']` only |
| Timestamp | `['entity:read']` only |

**Nested relation groups:** When a relation is exposed, the related entity's properties must include the parent's read group:

```php
// In Category entity, expose its fields when reading a VendorProfile:
#[Groups(['category:read', 'vendor:read'])]
private string $name = '';
```

## Security expression templates

```php
new GetCollection()  // no security — public listing
new Get()            // no security — public single resource
new Post(security: "is_granted('ROLE_VENDOR') or is_granted('ROLE_ADMIN')")
new Patch(security: "is_granted('entity:edit', object)")
new Delete(security: "is_granted('ROLE_ADMIN') or is_granted('entity:edit', object)")

// For private resources (e.g. WeddingPlan — only the owner can read it):
new Get(security: "is_granted('entity:view', object)")
new GetCollection(security: "is_granted('IS_AUTHENTICATED_FULLY')")
```

## Getter/setter convention

All getters and setters on one line when the body is trivial:

```php
public function getId(): ?int { return $this->id; }

public function getName(): string { return $this->name; }
public function setName(string $name): static { $this->name = $name; return $this; }

/** @return Collection<int, RelatedEntity> */
public function getRelatedItems(): Collection { return $this->relatedItems; }

public function addRelatedItem(RelatedEntity $item): static
{
    if (!$this->relatedItems->contains($item)) {
        $this->relatedItems->add($item);
    }
    return $this;
}

public function removeRelatedItem(RelatedEntity $item): static
{
    $this->relatedItems->removeElement($item);
    return $this;
}
```

Constructor must initialise all `Collection` properties:

```php
public function __construct()
{
    $this->relatedItems = new ArrayCollection();
}
```
